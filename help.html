Help Guide

We're here to help you get started!

How to Create a Decision

  1. Click "Create New Decision" on Home
  2. Enter a clear title and options
  3. Choose a selection mode: Vote or Random
  4. If using Vote mode, set tally target and max participants
  5. Click "Create Decision"

How to Join a Decision

  1. Click "Join Decision" on Home
  2. Enter your name
  3. Select an active vote decision with space available
  4. Click "Join Decision" to enter the voting room for that decision

Selection Modes

  • Vote: Participants cast one vote, and tallies are tracked.
  • Random: The system picks one option for quick decisions.

Using Active Decisions

In the Active Decisions section, you can:

  • Track participation and refreshed vote totals for vote decisions
  • Run random picks for random decisions
  • Use "Complete Decision" to move vote decisions to Completed
  • Keep random decisions available for repeated picks
